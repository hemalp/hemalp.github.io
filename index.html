The Waterloo Banking Project

The Waterloo Banking Project is a not-for-profit startup aiming to become Canada's first student run credit union. I am currently the Senior Director and I also assisted in the development of the web app which was built using Python, Django and PostgreSQL.

StoreTracker

StoreTracker was an app developed at Hack the North Hackathon. It's an iOS mobile application that sends JSON objectsd to a backend server developed with ruby-on-rails. The application recognizes Estimote-iBeacon's and collects distance and time data to implement and improve proximity based marketing.

Tetris

Developed a non-real time tetris application controlled through a command-line interface. The game was developed using C++ with X11 graphics. Used the Observer pattern and also coloured game pieces to enhance user experience.
